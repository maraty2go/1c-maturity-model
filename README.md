# Модель зрелости для команд 1С

Интерактивный инструмент для самооценки зрелости команд разработки на платформе 1С. Модель включает **8 ключевых направлений** и **32 практики**.

## Возможности
- Оценка по 32 практикам с детальными уровнями 1-5
- Радарная диаграмма профиля зрелости
- Автоматические рекомендации по росту
- Экспорт результатов в PDF и JSON
- Сохранение прогресса в браузере

## Как использовать
1. Скачайте `index.html` или клонируйте репозиторий
2. Откройте файл в любом современном браузере
3. Пройдите оценку по всем практикам
4. Изучите результаты и рекомендации

## Технологии
- HTML5, CSS3 (Tailwind CSS)
- JavaScript (Chart.js, html2canvas, jsPDF)
