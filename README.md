# Модель зрелости для команд 1С

![Версия](https://img.shields.io/badge/версия-1\.0-red)
![Практики](https://img.shields.io/badge/практики-32-blue)
![Лицензия](https://img.shields.io/badge/лицензия-MIT-green)

Интерактивный инструмент для самооценки зрелости команд разработки на платформе 1С. Модель разработана **Маратом Мустафиным**, архитектором стрима 1С в МТС, и включает **8 ключевых направлений (капабилити)** и **32 практики** с детальным описанием уровней 1–5.

---

## 📋 О модели

Модель зрелости оценивает не формальное соответствие общим корпоративным чек-листам, а реальную **глубину инженерной культуры** команды. Она учитывает специфику платформы 1С: работу с хранилищем, gitsync, расширения, Vanessa-Automation и другие особенности.

**8 направлений оценки:**
- Управление кодом и разработка
- Надёжность и наблюдаемость (SRE для 1С)
- Релизы и развертывание (CI/CD)
- Тестирование
- Архитектура и данные
- Инфраструктура и безопасность
- Продуктовый подход и процессы
- Искусственный интеллект и автоматизация знаний

---

## ✨ Возможности

- ✅ **32 практики** с детальными уровнями зрелости 1–5
- ✅ **Радарная диаграмма** профиля по 8 направлениям
- ✅ **Автоматические рекомендации** на основе слабых мест
- ✅ **Экспорт результатов** в PDF и JSON
- ✅ **Сохранение прогресса** в браузере (localStorage)
- ✅ **Полностью автономная работа** — один HTML-файл, не требует сервера

## 🚀 Как использовать

### Вариант 1: Онлайн (рекомендую)
Просто откройте [демо-версию](https://maraty2go.github.io/1c-maturity-model/) в браузере.

### Вариант 2: Локально
1. Скачайте файл [`index.html`](https://github.com/maraty2go/1c-maturity-model/raw/main/index.html)
2. Откройте его в любом современном браузере (Chrome, Firefox, Edge)
3. Последовательно оцените свою команду по каждой из 32 практик
4. Нажмите **«Результаты»** для анализа

### Вариант 3: Клонирование репозитория
```bash
git clone https://github.com/maraty2go/1c-maturity-model.git
cd 1c-maturity-model
# Откройте index.html в браузере
```

## 📊 Пример результатов

После завершения оценки вы увидите:
- **Радарную диаграмму** с профилем зрелости
- **Таблицу** всех оценок по практикам
- **Общий средний балл** по всем направлениям
- **Рекомендации** по развитию самых слабых областей

Результаты можно сохранить в JSON или экспортировать в PDF для отчёта руководству.
